About the Project

What is Learn Trade Grow?

An interactive web platform built to help users develop real investment knowledge. Eight structured lessons guide users from stock market basics to advanced trading strategies, each gated by a quiz with a 70% pass mark. Users then apply their knowledge in a simulator using live Alpaca Markets data, trading €10,000 of virtual funds in a risk-free environment.
